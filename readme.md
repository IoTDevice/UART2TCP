# UART2TCP
## 简介
 一个利用esp8266，实现串口到TCP，TCP到串口的数据转发的项目，可以理解为串口透传  
 ## 注意
 本项目是使用VS Code + Platform IO 来构建的，需要在其它IDE上编译的，请自行解决库的问题

> ## control with app(使用以下app操控)：
> * https://github.com/OpenIoTHub/OpenIoTHub
> * 此程序会自动发现设备，将设备放入智能设备列表

#### 支持的功能：
- [x] 1.与esp8266串口收发数据

#### 项目使用的硬件  
  * esp8266
